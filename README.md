# OpenTruth — Documentation

Bienvenue dans la documentation d'OpenTruth, l'infrastructure de verification automatisee du discours politique.

OpenTruth analyse les videos de discours politiques, debats parlementaires et interventions mediatiques pour extraire, verifier et noter les affirmations factuelles.

## Principe fondateur

> Nous n'evaluons pas le discours politique. Nous le structurons en donnees.
